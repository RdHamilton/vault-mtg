# Finance

Finance documents owned by `finance-controller`. See `reports/` for cost models.

## Scope

- AWS cost models and forecasts
- Subscription pricing models (post-GA)
- Beta-period cost-per-active-user analysis
- Vendor cost summaries (Clerk, Sentry, PostHog, Crisp, RDS, CloudFront)

## Owners

- `finance-controller` (primary)
- `business-analyst` (data inputs)

## Notes

This directory currently holds no published reports — finance work is tracked in private docs and surfaces here only when it informs an architecture or product decision (e.g., the Clerk Pro decision in v0.4.0 OQ-1).
